# CLI Chat Bot using JavaScript and OpenAI API

![Chat Bot](chat-bot-image.png)

This is a Command Line Interface (CLI) Chat Bot created using JavaScript and powered by the OpenAI API. The chat bot interacts with users through the terminal, providing a conversational experience using natural language processing capabilities from OpenAI.

## Features

- Real-time conversation: Engage in dynamic, back-and-forth conversations with the chat bot.
- Natural Language Processing: Leverages the OpenAI API to understand and generate human-like responses.
- Customizable settings: You can tweak certain parameters and configurations to suit your specific use case.

## Prerequisites

Before running the CLI chat bot, you need to have the following installed:

- Node.js: Ensure you have Node.js installed on your system.

1. Clone the repository from GitHub:

2. Install the dependencies:

3. Get your OpenAI API key:

   - Sign up for an account on the OpenAI website: https://openai.com
   - Retrieve your API key from the OpenAI dashboard.

4. Configure the API key:

   - Change the example API_KEY in the `.env` file in the project's root directory.

## Usage

To start the CLI chat bot, run the following command in your terminal:

"node script.js"

The chat bot will greet you and await your input. Type your messages and press `Enter` to send them to the chat bot. It will process your input using the OpenAI API and respond accordingly.

You can engage in a continuous conversation by taking turns with the chat bot. Press `Ctrl + Z` to exit the chat.
